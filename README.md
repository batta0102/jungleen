# jungleen

Monorepo for the JungleInEnglish project.

This repository contains multiple backend microservices and the Angular frontend template.

## Frontend

The frontend is located in `frontend/ProjetPI4eme-newTemplate/ProjetPI4eme-newTemplate`.

To start the frontend locally:

```bash
ng serve --proxy-config proxy.conf.json
```

## Backend

Each backend service has its own Maven wrapper and Spring Boot entry point under `backend/`.
