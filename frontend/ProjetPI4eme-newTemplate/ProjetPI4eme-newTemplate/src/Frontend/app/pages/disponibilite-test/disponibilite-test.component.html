💡 Instructions de débogage

1. Ouvrez la console du navigateur (F12) pour voir les logs détaillés

2. Vérifiez que votre backend Spring Boot est démarré sur http://localhost:9090

3. Testez les endpoints directement avec curl ou Postman:

curl http://localhost:9090/api/disponibilites/upcoming

curl http://localhost:9090/api/disponibilites/filter?filter=upcoming

curl http://localhost:9090/api/disponibilites/user/1

curl http://localhost:9090/api/disponibilites/buddy/10

4. Modifiez les IDs de test avec les champs ci-dessus pour tester différentes données

5. Vérifiez le proxy.conf.json pour vous assurer qu'il pointe vers localhost:9090
